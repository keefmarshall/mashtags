Mashtags

Manage your followed hashtags in Mastodon

You can follow hashtags in Mastodon. However, right now, there's no way in the core interface to manage them. This site will list all the tags you've followed, and allows you to delete any that you no longer find useful.

Hopefully the core web site and mobile apps will integrate this feature at some point soon and this site will become obsolete.

Privacy Statement

This application does not collect or store any of your data. It does temporarily request an access token on your behalf, which allows the Mastodon API interactions to retrieve your hashtags (read:follow) and to delete hashtags (write:follow). The access token is stored in a temporary session which is automatically deleted when the session expires. A necessary session cookie is set to retain application state between requests.
